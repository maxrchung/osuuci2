Map Pool

We have compiled a collection of the week's maps together for download. Inside the .rar is a copy of the osu!Collections Editor to interact with our .osdb collection. More indepth instructions are available in the Editor link, but for basic functionality:

  • Load your current beatmaps by pointing to your root osu! folder
  • Open our collection file
  • Search for and download beatmaps that you are missing
  • Join our collection file with your collection file
  • Save the joined collection file as your new osu! collections
